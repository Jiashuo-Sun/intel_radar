# Intel Radar — 产品设计文档

## 产品定位

Intel Radar 是一个**本地运行的行业情报系统**，服务于 Drawing AI / 延峰项目的商务与技术团队。

核心价值：**不是给你更多信息，而是每天早上告诉你"今天有什么值得关注的"。**

---

## 目标用户与场景

| 角色 | 使用方式 | 关注内容 |
|------|----------|----------|
| 商务（小湖） | 每天读日报 `.md` | 竞品融资/发布/中标、延峰动态、客户信号 |
| 技术负责人 | 周报技术专区 | 图像识别模型、OCR新论文、开源工具 |
| 管理层 | 周五周报综述 | 行业趋势、威胁/机会摘要 |

---

## 监控对象

### 精品监控（Premium Watch）

对指定公司深度跟踪：官网博客、新闻页、RSS。命中即推送。

| 公司 | 监控重点 | 优先级 |
|------|----------|--------|
| 延峰集团 (Yanfeng) | 官网新闻 + 公众号 | critical |
| 志丞科技 | 官网 + 融资动态 | high |
| Werk24 | 官网博客 + Google News | critical |
| Energent.ai | 官网 + Twitter/X | critical |
| High QA | 官网新闻页 | high |
| InspectionXpert / Ideagen | 官网 + 新闻稿 | high |
| CoLab Software | 官网博客 | normal |
| 1Factory | 官网新闻 | normal |
| DISCUS | 官网 | normal |

### 行业监控（Topic Watch）

通过 Google News RSS 和 arXiv API 按关键词广播，按打分过滤。

| 分组 | 典型关键词 |
|------|-----------|
| `drawing_ai` | engineering drawing OCR AI、工业图纸AI识别 |
| `yanfeng_ecosystem` | 延峰数字化、PPAP automation、IATF16949 |
| `tech_frontier` | multimodal OCR industrial、document understanding model |
| `market_signals` | 工业数字化招标、智能制造政府采购 |

## 打分逻辑

每条情报按关键词规则打分（0-10分），分数决定在日报中的位置：

| 分数 | 日报区域 | AI摘要 |
|------|----------|--------|
| ≥ 4 | 🔴 高优先级（今日必读） | ✅ 是 |
| 2-3 | 🟡 行业动态 / 🔵 竞品动态 | ✅ 是 |
| 1 | 🟡 行业动态 | ✅ 是 |
| 0 | 过滤，不出现在日报 | ❌ 否 |

高分信号词：融资(+3)、中标(+3)、并购(+3)、Werk24/延峰(+3)、产品发布(+2)

---

## 日报输出格式

每天生成 `output/YYYY-MM-DD.md`，包含：

1. **🔴 高优先级**：详细展开，含 AI 摘要 + 影响判断 + 建议行动
2. **🔵 竞品动态**：精品监控中分数在 2-3 的条目，表格形式
3. **🟡 行业动态**：其他中等分数条目，按关键词分组
4. **📚 技术前沿**：arXiv 新论文列表
5. **统计行**：已见过 / 低相关过滤 / 近似去重
